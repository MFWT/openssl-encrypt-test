# openssl-encrypt-test
调用OpenSSL，测试各加密算法的性能  
Call OpenSSL to test the performance of each encryption algorithm

## Usage
常用测试（仅测试AES-128-GCM、AES-256-GCM，Chacha20-IETF-Poly1305）
```shell
./openssl-speed-test
```

完整测试
```shell
./openssl-speed-test full
```
